GitHub 文件加速

快速、稳定地获取 GitHub 资源

使用说明
  • GitHub 文件链接带不带协议头都可以，支持 release、archive 以及文件
  • release、archive 使用 Cloudflare 加速，文件会跳转至 JsDelivr
  • 右键复制出来的链接都是符合标准的，注意不支持项目文件夹
合法输入示例
https://github.com/dstansice/project/archive/master.zip点击复制
https://github.com/dstansice/project/archive/v0.1.0.tar.gz点击复制
https://github.com/dstansice/project/releases/download/v0.1.0/example.zip点击复制
https://github.com/dstansice/project/blob/master/filename点击复制
